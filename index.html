Presentation Details

This project is a presentation on ray tracing. We go through a brief introduction to the model, and compare it to conventional rasterized graphics. After seeing the advantages of ray tracing, we go through the process of creating a ray tracer from scratch. We start with a simple ray tracer - with a single ray, and then add more features to it. These features include a more complex voxel-based scene, additional rays, and various levels of reflection with a recursive ray tracer. We will briefly overview the pseudocode for such a ray tracer. Lastly, we'll explore the differences in eye-based and light-based ray tracing, the use cases of both of them, and the advantages and disadvantages of each. This is a very brief overview and does not fully encapsulate the details of this presentation. It is here as a placeholder for more text.
